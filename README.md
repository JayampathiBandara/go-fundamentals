- git commit -m "Add demo for the four data types" → commits locally
- git remote add origin (https://github.com/JayampathiBandara/go-fundamentals.git) → just saves remote URL
- git branch -M main → renames local branch
- git push -u origin main → this is the first command that contacts GitHub
